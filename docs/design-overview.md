# 黄金矿工 H5 - 设计概述

> 本文档是 `plans/game-design.md` 的精简索引，供开发时快速查阅。完整设计细节请查看原始设计文档。

## 游戏定位

经典黄金矿工像素风复刻，纯 HTML5 Canvas + TypeScript 实现，**零图片资源依赖**，所有像素素材通过代码数据矩阵（PixelMap）生成。

## 核心玩法循环

```
钩爪摆动 → 玩家点击发射 → 碰到矿物自动收回（速度受重量影响）
    ↑                                    │
    └────────── 收回完成，计分 ◄──────────┘
```

## 关键公式

| 公式 | 表达式 |
|------|--------|
| 钩爪摆动角度 | `angle = sin(time * swingSpeed) * maxAngle` |
| 钩爪发射方向 | `dx = cos(angle) * speed`, `dy = sin(angle) * speed` |
| 收回速度 | `baseSpeed / (1 + weight * 0.5)` |
| 碰撞判定 | 钩爪尖端到矿物中心距离 < 矿物包围圆半径 |

## 设计参数

- **逻辑分辨率：** 480 x 640（竖屏）
- **像素精灵尺寸：** 32x32 标准格

- **配色基调：** 天空蓝 + 土壤棕 + 金色点缀
